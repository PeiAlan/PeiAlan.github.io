JDK1.8дёӯзҡ„ConcurrentHashMap

жңҖеҗҺйҷ„дёҠеёёи§Ғзҡ„иҝҷеқ—зҡ„йқўиҜ•йўҳпјҹ

1гҖҒHashMap е’Ң HashTable жңүд»Җд№ҲеҢәеҲ«пјҹ
      в‘ гҖҒHashMap жҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„пјҢHashTable жҳҜзәҝзЁӢе®үе…Ёзҡ„пјӣ
      в‘ЎгҖҒз”ұдәҺзәҝзЁӢе®үе…ЁпјҢжүҖд»Ҙ HashTable зҡ„ж•ҲзҺҮжҜ”дёҚдёҠ HashMapпјӣ
      в‘ўгҖҒHashMap жңҖеӨҡеҸӘе…Ғи®ёдёҖжқЎи®°еҪ•зҡ„й”®дёә nullпјҢе…Ғи®ёеӨҡжқЎи®°еҪ•зҡ„еҖјдёә nullпјҢ иҖҢ HashTable дёҚе…Ғи®ёпјӣ
      в‘ЈгҖҒHashMap й»ҳи®ӨеҲқе§ӢеҢ–ж•°з»„зҡ„еӨ§е°Ҹдёә 16пјҢHashTable дёә 11пјҢеүҚиҖ…жү©е®№ж—¶пјҢ жү©еӨ§дёӨеҖҚпјҢеҗҺиҖ…жү©еӨ§дёӨеҖҚ+1пјӣ
      в‘ӨгҖҒHashMap йңҖиҰҒйҮҚж–°и®Ўз®— hash еҖјпјҢиҖҢ HashTable зӣҙжҺҘдҪҝз”ЁеҜ№иұЎзҡ„ hashCode

2гҖҒJava дёӯзҡ„еҸҰдёҖдёӘзәҝзЁӢе®үе…Ёзҡ„дёҺ HashMap жһҒе…¶зұ»дјјзҡ„зұ»жҳҜд»Җд№ҲпјҹеҗҢж ·жҳҜзәҝзЁӢе®үе…ЁпјҢе®ғдёҺ HashTable еңЁзәҝзЁӢеҗҢжӯҘдёҠжңүд»Җд№ҲдёҚеҗҢпјҹ
      ConcurrentHashMap зұ»пјҲжҳҜ Java е№¶еҸ‘еҢ… java.util.concurrent дёӯжҸҗдҫӣзҡ„дёҖ дёӘзәҝзЁӢе®үе…Ёдё”й«ҳж•Ҳзҡ„ HashMap е®һзҺ°пјүгҖӮ
      HashTable жҳҜдҪҝз”Ё synchronize е…ій”®еӯ—еҠ й”Ғзҡ„еҺҹзҗҶпјҲе°ұжҳҜеҜ№еҜ№иұЎеҠ й”Ғпјүпјӣ иҖҢй’ҲеҜ№ ConcurrentHashMapпјҢеңЁ JDK 1.7 дёӯйҮҮз”ЁеҲҶж®өй”Ғ(SegmentеҶ…йғЁзұ»)зҡ„ж–№ејҸпјӣJDK 1.8 дёӯ зӣҙжҺҘйҮҮз”ЁдәҶ CASпјҲж— й”Ғз®—жі•пјү+ synchronizedпјҢд№ҹйҮҮз”ЁеҲҶж®өй”Ғзҡ„ж–№ејҸе№¶еӨ§еӨ§зј©е°ҸдәҶ й”Ғзҡ„зІ’еәҰгҖӮ

3гҖҒHashMap & ConcurrentHashMap зҡ„еҢәеҲ«пјҹ
      йҷӨдәҶеҠ й”ҒпјҢеҺҹзҗҶдёҠж— еӨӘеӨ§еҢәеҲ«гҖӮ еҸҰеӨ–пјҢHashMap зҡ„й”®еҖјеҜ№е…Ғи®ёжңү nullпјҢдҪҶжҳҜ ConCurrentHashMap йғҪдёҚе…Ғи®ёгҖӮ еңЁж•°жҚ®з»“жһ„дёҠпјҢзәўй»‘ж ‘зӣёе…ізҡ„иҠӮзӮ№зұ»

4гҖҒдёәд»Җд№Ҳ ConcurrentHashMap жҜ” HashTable ж•ҲзҺҮиҰҒй«ҳпјҹ
      HashTable дҪҝз”ЁдёҖжҠҠй”ҒпјҲй”ҒдҪҸж•ҙдёӘй“ҫиЎЁз»“жһ„пјүеӨ„зҗҶе№¶еҸ‘й—®йўҳпјҢеӨҡдёӘзәҝзЁӢ з«һдәүдёҖжҠҠй”ҒпјҢе®№жҳ“йҳ»еЎһпјӣ ConcurrentHashMap JDK 1.7 дёӯдҪҝз”ЁеҲҶж®өй”ҒпјҲReentrantLock + Segment + HashEntryпјүпјҢзӣёеҪ“дәҺжҠҠдёҖ дёӘ HashMap еҲҶжҲҗеӨҡдёӘж®өпјҢжҜҸж®өеҲҶй…ҚдёҖжҠҠй”ҒпјҢиҝҷж ·ж”ҜжҢҒеӨҡзәҝзЁӢи®ҝй—®гҖӮй”ҒзІ’еәҰпјҡеҹә дәҺ SegmentпјҢеҢ…еҗ«еӨҡдёӘ HashEntryгҖӮ JDK 1.8 дёӯдҪҝз”Ё CAS + synchronized + Node + зәўй»‘ж ‘гҖӮй”ҒзІ’еәҰпјҡNodeпјҲйҰ–з»“ зӮ№пјүпјҲе®һзҺ° Map.Entry<K,V>пјүгҖӮй”ҒзІ’еәҰйҷҚдҪҺдәҶгҖӮ

5гҖҒConcurrentHashMap й”ҒжңәеҲ¶е…·дҪ“еҲҶжһҗпјҲJDK 1.7 VS JDK 1.8пјүпјҹ
      JDK 1.7 дёӯпјҢйҮҮз”ЁеҲҶж®өй”Ғзҡ„жңәеҲ¶пјҢе®һзҺ°е№¶еҸ‘зҡ„жӣҙж–°ж“ҚдҪңпјҢеә•еұӮйҮҮз”Ёж•°з»„+й“ҫиЎЁ зҡ„еӯҳеӮЁз»“жһ„пјҢеҢ…жӢ¬дёӨдёӘж ёеҝғйқҷжҖҒеҶ…йғЁзұ» Segment е’Ң HashEntryгҖӮ
      в‘ гҖҒSegment з»§жүҝ ReentrantLockпјҲйҮҚе…Ҙй”Ғпјү з”ЁжқҘе……еҪ“й”Ғзҡ„и§’иүІпјҢжҜҸдёӘ Segment еҜ№иұЎе®ҲжҠӨжҜҸдёӘж•ЈеҲ—жҳ е°„иЎЁзҡ„иӢҘе№ІдёӘжЎ¶пјӣ
      в‘ЎгҖҒHashEntry з”ЁжқҘе°ҒиЈ…жҳ е°„иЎЁзҡ„й”®-еҖјеҜ№пјӣ
      в‘ўгҖҒжҜҸдёӘжЎ¶жҳҜз”ұиӢҘе№ІдёӘ HashEntry еҜ№иұЎй“ҫжҺҘиө·жқҘзҡ„й“ҫиЎЁгҖӮ JDK 1.8 дёӯпјҢйҮҮз”Ё Node + CAS + Synchronized жқҘдҝқиҜҒе№¶еҸ‘е®үе…ЁгҖӮеҸ–ж¶Ҳзұ» SegmentпјҢзӣҙжҺҘз”Ё table ж•°з»„еӯҳеӮЁй”®еҖјеҜ№пјӣеҪ“ HashEntry еҜ№иұЎз»„жҲҗзҡ„й“ҫиЎЁй•ҝеәҰи¶… иҝҮ TREEIFY_THRESHOLD ж—¶пјҢй“ҫиЎЁиҪ¬жҚўдёәзәўй»‘ж ‘пјҢжҸҗеҚҮжҖ§иғҪгҖӮеә•еұӮеҸҳжӣҙдёәж•°з»„ + й“ҫиЎЁ + зәўй»‘ж ‘гҖӮ

6гҖҒConcurrentHashMap еңЁ JDK 1.8 дёӯпјҢдёәд»Җд№ҲиҰҒдҪҝз”ЁеҶ…зҪ®й”Ғ synchronized жқҘд»ЈжӣҝйҮҚе…Ҙй”Ғ ReentrantLockпјҹ
      1гҖҒJVM ејҖеҸ‘еӣўйҳҹеңЁ 1.8 дёӯеҜ№ synchronized еҒҡдәҶеӨ§йҮҸжҖ§иғҪдёҠзҡ„дјҳеҢ–пјҢиҖҢдё”еҹә дәҺ JVM зҡ„ synchronized дјҳеҢ–з©әй—ҙжӣҙеӨ§пјҢжӣҙеҠ иҮӘз„¶гҖӮ
      2гҖҒеңЁеӨ§йҮҸзҡ„ж•°жҚ®ж“ҚдҪңдёӢпјҢеҜ№дәҺ JVM зҡ„еҶ…еӯҳеҺӢеҠӣпјҢеҹәдәҺ API зҡ„ ReentrantLock дјҡејҖй”ҖжӣҙеӨҡзҡ„еҶ…еӯҳгҖӮ

7гҖҒ1.8дёӢConcurrentHashMap з®ҖеҚ•д»Ӣз»Қпјҹ
      в‘ гҖҒйҮҚиҰҒзҡ„еёёйҮҸпјҡ private transient volatile int sizeCtl; еҪ“дёәиҙҹж•°ж—¶пјҢ-1 иЎЁзӨәжӯЈеңЁеҲқе§ӢеҢ–пјҢ-N иЎЁзӨә N - 1 дёӘзәҝзЁӢжӯЈеңЁиҝӣиЎҢжү©е®№пјӣ еҪ“дёә 0 ж—¶пјҢиЎЁзӨә table иҝҳжІЎжңүеҲқе§ӢеҢ–пјӣ еҪ“дёәе…¶д»–жӯЈж•°ж—¶пјҢиЎЁзӨәеҲқе§ӢеҢ–жҲ–иҖ…дёӢдёҖж¬ЎиҝӣиЎҢжү©е®№зҡ„еӨ§е°ҸгҖӮ
      в‘ЎгҖҒж•°жҚ®з»“жһ„пјҡ Node жҳҜеӯҳеӮЁз»“жһ„зҡ„еҹәжң¬еҚ•е…ғпјҢз»§жүҝ HashMap дёӯзҡ„ EntryпјҢз”ЁдәҺеӯҳеӮЁж•°жҚ®пјӣ TreeNode з»§жүҝ NodeпјҢдҪҶжҳҜж•°жҚ®з»“жһ„жҚўжҲҗдәҶдәҢеҸүж ‘з»“жһ„пјҢжҳҜзәўй»‘ж ‘зҡ„еӯҳеӮЁ з»“жһ„пјҢз”ЁдәҺзәўй»‘ж ‘дёӯеӯҳеӮЁж•°жҚ®пјӣ TreeBin жҳҜе°ҒиЈ… TreeNode зҡ„е®№еҷЁпјҢжҸҗдҫӣиҪ¬жҚўзәўй»‘ж ‘зҡ„дёҖдәӣжқЎд»¶е’Ңй”Ғзҡ„жҺ§еҲ¶гҖӮ
